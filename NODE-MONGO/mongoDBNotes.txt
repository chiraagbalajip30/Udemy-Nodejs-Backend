Introduction to MongoDB (NOSQL)


We are going to explore MongoDB which is a NOSQL DB.

Mongoose is actually an ODM (Object Document Mapper) instead of an ORM.

NOSQL DB:
---------
1. It is an opposite to SQL DB. 
2. In SQL DB, we store data in a structured or table manner. For ex, in the previous usge of example we had used users table, etc. 
Most importantly, SQL has relations. For example, userSSessions has relation with the users table. All the rules are applied on the database-level. 
3. Whereas, there are no rules in the NOSQl DB. All the rules are applied on the application-level that is, Node.js level. We have to validate the rules and everything. There are no rules, no tables nothing. They donot have any structure. 
4. There are something known as Documents in NOSQL. Data are stored in this format. For example, in QL, we had a column with columnname, with a particular id and everything. Whereas in NOSQL, for example, in the same table, ione column can have id and name, whereas the other column in the same table can have fname and id. The structure of one entry can be different from the other. This means there is no Schema. SO to manage this, we have to do it in the application that is, Node.js. 
5. Examples of DB:
SQL DB - Postgres, MySQL, etc.
NOSQL (Document DB's) - MongoDB, AWS Dynamo DB, etc.

The only advantage of NOSQLDB is there is more dynamicity that is, more flexiblity.

import express from "express";
import { User } from "../models/user.model.js";
import { randomBytes, createHmac } from "node:crypto";
import jwt from "jsonwebtoken";

const router = express.Router();

router.post("/signup", async (req, res) => {
  const { name, email, password } = req.body;

  const existingUser = await User.findOne({
    email,
  });

  if (existingUser) {
    return res
      .status(400)
      .json({ error: `User with email ${email} already exists` });
  }

  const salt = randomBytes(256).toString("hex");
  const hashedPassword = createHmac("sha256", salt)
    .update(password)
    .digest("hex");

  const user = await User.insertOne({
    name,
    email,
    password: hashedPassword,
    salt,
  });

  return res.status(201).json({ status: "Success", data: { id: user._id } });
});

router.post("/login", async (req, res) => {
  const { email, password } = req.body;

  const existingUser = await User.findOne({
    email,
  });

  if (!existingUser) {
    return res
      .status(400)
      .json({ error: `User with email ${email} does not exist` });
  }

  const salt = existingUser.salt;
  const hashed = existingUser.password;

  const newHash = createHmac("sha256", salt).update(password).digest("hex");

  if (hashed !== newHash) {
    return res.status(400).json({ error: "Incorrect Password " });
  }

  const payload = {
    _id: existingUser._id,
    name: existingUser.name,
    email: existingUser.email,
  };

  const token = jwt.sign(payload, process.env.JWT_SECRET);

  return res.status(200).json({ status: "Success", token });
});

export default router;
